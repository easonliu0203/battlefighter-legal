# BATTLEFIGHTER — Legal

Public-facing **Privacy Policy** and **Terms of Service** pages for the BATTLEFIGHTER app.

These are static pages that fetch content live from the app's `legal_documents` table
(public read-only), so updating the policy in-app automatically updates these pages —
no redeploy needed. Languages: 中文 / 日本語 / English.

- Privacy Policy — `privacy.html`
- Terms of Service — `terms.html`

Hosted via GitHub Pages. No application source code lives in this repository.
